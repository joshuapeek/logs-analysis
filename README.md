This python script connects to the 'news' database, and runs three queries, to satisfy the requirements of the assignment.

To initially launch the script, enter the following within the terminal:
python project_logs.py

You should see a message printed within the terminal, confirming that the queries are being run, and shortly thereafter, you should see the answers displayed.

For each question, you will see the text of the question displayed, and a sorted list with the "answers" displayed below.
